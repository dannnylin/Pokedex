# Pokedex

<br/>
<h5>Skills Practiced:</h5>
<ul>
<li>UIStackView</li>
<li>UICollectionView</li>
<li>UISearchBar</li>
<l>Utilizing frameworks (Alamofire)</li>
<li>Using APIs (PokeAPI)</li>
<li>Implementing MVC</li>
<li>JSON parsing</li>
<li>CSV parsing</li>
</ul>
